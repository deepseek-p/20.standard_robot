## 14:57:02 | 连杆机构分析 + 前馈方向修正 + 角度范围测量 + 自适应前馈方案确定

### 参数变更

**调前:**
```
pitch_encode Kp=24 Ki=0 Kd=0 max_out=10 max_iout=0.5
pitch_speed Kp=800 Ki=10 Kd=0 max_out=15000 max_iout=3000
PITCH_GRAVITY_K_FIXED=600, K_AMMO_FULL=4000, AMMO_CAPACITY=300
前馈方向: PITCH_TURN时 current_set -= compensation (错误)
```

**调后:**
```
前馈方向已修正: PITCH_TURN时 current_set += compensation
K_FIXED=600 + K_AMMO_FULL=4000 对空仓过大(总4600)，需改为自适应方案
PID参数未变
```

### 效果对比

1. 前馈方向测试:
   - 错误方向(current_set -= compensation): 空仓pitch_rel=-0.24, set=-0.17, 误差0.07rad, cur=-195 (前馈帮倒忙)
   - 修正方向(current_set += compensation): pitch_rel=-0.01, set=-0.05, 误差0.04rad, cur=821 (方向正确但K过大)

2. 连杆机构分析:
   - 四连杆参数: 上杆42mm, 下杆42mm, 连杆131mm, 机架131mm
   - 上下杆等长+连杆机架等长 = 标准平行四边形, 传动比1:1
   - 电机轴角度 = pitch实际俯仰角度

3. 角度范围测量(空仓,手动推到限位):
   - 上限位(最抬头): pitch_rel=-0.21, cur=+7782
   - 下限位(最低头): pitch_rel=+0.18, cur=-5765
   - 总范围: 0.39rad ≈ 22°

4. K*cosθ模型失效分析:
   - cos(-0.21)=0.978, cos(0.18)=0.984, 差仅0.6%
   - 但保持电流差: 7782 vs -5765, 差13547
   - 等效K在不同角度差异巨大, 说明摩擦/线束力矩随角度变化显著
   - 单一K*cosθ前馈不够, 需要自适应方案

5. 自适应方案确定:
   - 采用LMS在线自适应: K_hat和b_hat在准静态时自动更新
   - 模型: I_ff = K_hat*cos(θ+φ) + b_hat
   - 准静态条件: |speed|<阈值, |angle_error|<阈值, 电流未饱和
   - 优势: 不需要弹量传感器, 自动适应空仓到满仓全范围

### 证据文件

- [data/tune_2026-02-27_014.csv](../../data/tune_2026-02-27_014.csv)
- [data/tune_2026-02-27_015.csv](../../data/tune_2026-02-27_015.csv)
- [data/tune_2026-02-27_016.csv](../../data/tune_2026-02-27_016.csv)

### 结论

1. 前馈方向已修正(PITCH_TURN时用+=)
2. 连杆为平行四边形1:1传动比, pitch实际范围22°
3. 固定K*cosθ前馈不适用, 需改为在线自适应
4. 下一步: 实现LMS自适应前馈(K_hat+b_hat), 替换当前固定K方案
5. 弹丸计数方案废弃(初始弹量不确定), 改用电流自估计

## 15:59:43 | 自适应前馈空仓通过，装弹后死锁——准静态角度误差条件过严导致无法更新

### 参数变更

**调前:**
```
PITCH_FF_GAMMA_K=0.0005, GAMMA_B=0.0005
PITCH_FF_ERR_TH=0.02, SPEED_TH=0.5, SAT_TH=15000
PITCH_FF_ALPHA_DOT_MAX=5.0
```

**调后:**
```
待修改: 去掉angle_err条件, GAMMA提高到0.005, ALPHA_DOT_MAX提高到50
```

### 效果对比

空仓自适应前馈验证:
- 下限位(低头): rel=0.19, set=0.19, err=0, cur=1796
- 上限位(抬头): rel=-0.20, set=-0.20, err=0, cur=1417
- 前馈已收敛，两方向到位

装满弹后测试:
- rc_ch3=-660(摇杆向下拉满), pitch_mode=2
- rel=-0.17, set=0.19, err=0.36rad, cur=10330
- pitch纹丝不动，前馈不够但无法更新(angle_err>ERR_TH)
- 死锁: 前馈不够→推不动→误差大→不更新→前馈不够

### 证据文件

- [data/tune_2026-02-27_018.csv](../../data/tune_2026-02-27_018.csv)
- [data/tune_2026-02-27_020.csv](../../data/tune_2026-02-27_020.csv)
- [data/tune_2026-02-27_021.csv](../../data/tune_2026-02-27_021.csv)

### 结论

1. 自适应前馈空仓验证通过：两方向误差=0，电流均匀
2. 装弹后出现死锁：误差0.36rad远超ERR_TH(0.02)，自适应更新被锁死，pitch推不动
3. 根因：准静态条件中angle_err<0.02过于严格，被重力压住不动时速度≈0但误差大，应视为"静态"
4. 修复方案：去掉角度误差条件，加大学习率10倍(0.0005→0.005)，加大变化率限制(5→50)
